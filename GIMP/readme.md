# GIMP

## Keyboard shortcuts guide:


#### File
``` Ctrl + O ``` : Open

``` Ctrl + N ``` : New

``` Ctrl + Alt + O ``` : Open as layer go ``` Graphics ```, and select ```Gnu Image Manipulation Program```
